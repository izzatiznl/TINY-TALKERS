# Tiny Talkers

**Tiny Talkers** is an interactive educational app designed for children to learn basic English vocabulary in a fun and engaging way. Developed using **MIT App Inventor**, the app combines visual learning, voice pronunciation, and interactive quizzes to help kids improve their language skills.

## Key Features
- **Learn Basic Vocabulary:** Colors, numbers, fruits, and everyday objects.
- **Voice Pronunciation:** Kids can tap to hear how words are pronounced.
- **Interactive Quiz:** Fun quizzes to reinforce learning.
- **Shake Interaction:** Shake the device to change images into number characters for vocabulary practice.
- **User-Friendly Interface:** Colorful and easy-to-navigate screens suitable for young learners.
